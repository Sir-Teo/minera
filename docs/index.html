Minerva

Lightweight Multi-Physics Simulation Engine

A scalable simulation engine designed for Mac, covering macro-scale (rigid bodies, fluids) and micro-scale (molecular dynamics) physics with a modular architecture.

Try Playground View on GitHub

Key Features

Modular Architecture

Physics modules plug into a flexible scheduler system for easy extensibility

Multi-Scale Physics

Handle both macro (rigid bodies) and micro (molecular dynamics) physics in a unified framework

High Performance

Cache-friendly data structures with O(N) neighbor lists achieving 4-5× speedup

Lightweight

Minimal dependencies, CPU-first design optimized for Mac (Apple Silicon/Intel)

Visualization Ready

CSV and VTK output for ParaView visualization and analysis

Research Grade

Sanitizer-clean code with rigorous validation and correctness testing

Quick Start

C++ 
// Create world and add physics systems
World world;

RigidBodySystemConfig rb_cfg;
rb_cfg.restitution = 0.5;
rb_cfg.ground_y = 0.0;

MDConfig md_cfg;
md_cfg.rcut_sigma = 2.5;

world.scheduler.add(std::make_unique<RigidBodySystem>(rb_cfg), 1);
world.scheduler.add(std::make_unique<MDSystem>(md_cfg), 1);

// Run simulation
const double dt = 1.0/120.0;
for (int i = 0; i < steps; ++i) {
    world.step(dt);
}

bash

ninja -C build
